# ViewNoa

VS Code起動時に、指定フォルダの画像をランダムに表示する拡張機能。

## Features

- VS Code起動時に自動的にWebviewを開き、設定されたフォルダの画像をランダム表示。
- 1分ごとに自動的に画像を変更。

## Requirements

- VS Code 1.107.0以上

## Extension Settings

この拡張機能には以下の設定があります：

- `viewNoa.imageFolder`: 画像を表示するフォルダのパスを指定してください。例: `C:\Users\YourName\Pictures\Screenshots`

設定方法：
1. VS Codeの設定を開く（`Ctrl + ,`）。
2. 「ViewNoa」と検索。
3. `ViewNoa: Image Folder` の項目にフォルダパスを入力。
